Speech-to-text — Whisper
A Whisper server, e.g. the onerahmet/openai-whisper-asr-webservice docker image (port 9000). OpenAI-compatible servers (speaches / faster-whisper-server) and whisper.cpp also work — the endpoint style is detected automatically.
Language model — Ollama
An Ollama model that supports tool calling (e.g. qwen3, llama3.1, gpt-oss). Empty = the first model from ollama list.
Text-to-speech — Piper
A Piper HTTP server (port 5000), e.g. artibex/piper-http or python -m piper.http_server. The spoken voice is the model the server was started with. No authentication is used — keep these services on your trusted LAN.
